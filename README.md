# သူရစားသောက်ဆိုင် POS (Point of Sale) System

မြန်မာဘာသာဖြင့် သုံးစွဲနိုင်သော စားသောက်ဆိုင်သုံး POS စနစ်ဖြစ်ပါသည်။

## 🌟 အဓိကပါဝင်သော Features များ (Core Features)

- **အရောင်းစနစ် (Dynamic POS Interface)**:
  - ကုန်ပစ္စည်းများကို အမျိုးအစားအလိုက် လွယ်ကူစွာ ရှာဖွေရောင်းချနိုင်သည်။
  - ခြင်း (Cart) ထဲသို့ ပစ္စည်းများထည့်ခြင်း၊ အရေအတွက် ပြင်ဆင်ခြင်းနှင့် ပယ်ဖျက်ခြင်းများကို စက္ကန့်ပိုင်းအတွင်း လုပ်ဆောင်နိုင်သည်။
  - အခွန် (Tax) ၅% ကို အလိုအလျောက် တွက်ချက်ပေးခြင်း။
- **ဘောင်ချာစနစ် (Invoice & Printing)**:
  - အရောင်းပြီးဆုံးပါက ဘောင်ချာ (Invoice) ကို အကျဉ်းချုပ်ပြသပေးခြင်း။
  - Thermal Printer များဖြင့် ဘောင်ချာထုတ်ယူရန် Print ထုတ်နိုင်သော Layout ပါဝင်ခြင်း။

- **ကုန်ပစ္စည်းစီမံခန့်ခွဲမှု (Product & Category Management)**:
  - စားသောက်ဖွယ်ရာများနှင့် အမျိုးအစားများကို စိတ်ကြိုက် ထည့်သွင်း၊ ပြင်ဆင်၊ ဖျက်သိမ်းနိုင်သည်။
- **အသုံးပြုသူစီမံခန့်ခွဲမှု (User Management)**:
  - ဝန်ထမ်းများ သို့မဟုတ် Admin များအတွက် Account များ သီးခြား ဖန်တီးနိုင်သည်။
  - လုံခြုံစိတ်ချရသော Login စနစ်နှင့် Protected Routes များ ပါဝင်သည်။

- **စာရင်းချုပ်အစီရင်ခံစာ (Reports)**:
  - နေ့စဉ်၊ လစဉ် အရောင်းစာရင်းများကို ပြန်လည်ကြည့်ရှုနိုင်သည်။

- **Responsive Design**:
  - ဖုန်း၊ တက်ဘလက် နှင့် ကွန်ပျူတာ အားလုံးတွင် အဆင်ပြေစွာ အသုံးပြုနိုင်သော Layout (Fully Responsive)။

---

## 🛠 နည်းပညာများ (Tech Stack)

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS (Modern & Clean Design)
- **Database & Auth**: Supabase
- **State Management**: TanStack Query (React Query)
- **Icons**: Lucide React
- **Notifications**: Sonner

---

## 🔄 လုပ်ငန်းစဉ်အဆင့်ဆင့် (Work Flow)

1. **Login**: အသုံးပြုသူသည် မိမိ၏ Email နှင့် Password ဖြင့် စနစ်ထဲသို့ ဝင်ရောက်ရမည်။
2. **Dashboard/POS**: Login ဝင်ပြီးနောက် အရောင်းစာမျက်နှာသို့ ရောက်ရှိမည်။
3. **Selling Process**:
   - ကုန်ပစ္စည်းများကို ရွေးချယ်၍ Cart ထဲသို့ ထည့်မည်။
   - "ငွေရှင်းမည်" (Checkout) ကို နှိပ်မည်။
4. **Checkout & Tax**: စနစ်မှ အခွန် ၅% ကို အလိုအလျောက် ပေါင်းထည့်ပေးမည်ဖြစ်ပြီး Supabase Database ထဲသို့ အရောင်းစာရင်း သိမ်းဆည်းမည်။
5. **Invoice**: အရောင်းအောင်မြင်ပါက ဘောင်ချာ ပြသမည်ဖြစ်ပြီး လိုအပ်ပါက Print ထုတ်နိုင်သည်။
6. **Data Management**: Admin များအနေဖြင့် Sidebar မှတစ်ဆင့် ကုန်ပစ္စည်းများ၊ အမျိုးအစားများနှင့် အသုံးပြုသူများကို လိုအပ်သလို စီမံခန့်ခွဲနိုင်သည်။
